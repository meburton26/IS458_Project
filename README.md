# NextPlay (NXP)

NextPlay (NXP) is an all-in-one sport analytics dashboard designed to give fans and analysts a strategic edge. The platform provides player-focused insights, trend analysis, matchup breakdowns, and quick-action metrics to help users make informed decisions for fantasy sports, trades, and betting.

## Features

- Player performance insights and statistics
- Matchup analysis for upcoming games
- Trend tracking for players and teams
- Quick metrics for fast decision-making
- Sports news and updates
- User account personalization
- Search functionality

## Pages

1. **Dashboard (dashboard.html)**
   Displays an overview of key stats, trending players, and highlights.
2. **Player Insights (player_insights.html)**
   Provides detailed player statistics and performance analysis.
3. **Matchups (matchup.html)**
   Breaks down upcoming games and compares teams and players.
4. **Trends (trends.html)**
   Shows performance trends over time.
5. **Quick Metrics (metrics.html)**
   Offers fast, actionable data for decisions.
6. **News (news.html)**
   Displays recent sports news and updates.
7. **Account (account.html)**
   Allows users to manage preferences and saved content.
8. **Help (help.html)**
   Provides guidance on how to use the website.

## Technologies Used

- HTML
- CSS
- JavaScript

## Navigation

The website includes a responsive navigation bar with links to all 8 pages, a search bar, and a mobile-friendly layout.
